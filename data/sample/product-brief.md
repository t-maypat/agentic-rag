Product vision:
We focus on clarity, speed, and trust. Users can see every source chunk used to build an answer, along with a trace of the agent steps.

Target users:
- AI engineers evaluating RAG quality and system behavior
- Product teams needing transparent, grounded answers
- Recruiters and hiring managers reviewing portfolio projects

Core UX principles:
- Single conversation view with zero context switching
- Side panel for sources with quick jump links
- Response format optimized for scanning (bullets + citations)
- Trace view that shows retrieve and synthesize phases

Key product requirements:
- Sub-2 second retrieval for top-k chunks
- Deterministic prompts for repeatable answers
- Highlight missing evidence when sources are insufficient

Roadmap ideas:
- Add dataset evaluation dashboard with RAGAS metrics
- Support hybrid search tuning per query
- Add lightweight auth for multi-user demos
